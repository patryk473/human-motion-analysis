1. Środowisko wirtualne
python -m venv venv
venv\Scripts\Activate.ps1

2. Zainstalowane pakiety
numpy → obliczenia wektorowe i macierze
scipy → interpolacja, filtrowanie, detekcja pików
matplotlib → wykresy, wizualizacja
pandas → zapis/odczyt CSV, analiza danych
opencv-python → przetwarzanie wideo i obrazów
mediapipe → wykrywanie landmarków / analiza ruchu

3a. Konfiguracja VS Code
Ctrl Shift P -> Select Interpreter -> venv

3b. Konfifuracja python
By mediapipe działał potrzebny był Python 3.10.11, bo 3.12 + nie wspiera
